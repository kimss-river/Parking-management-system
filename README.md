# Parking-management-system
   
* 주차 및 출차를 관리할 수 있다.
* 주차공간에 주차된 차가 있는지 조회한다.
* 공간번호에 지상, 층, 구역, 번호를 나눠 표시한다. 
* 주차, 출차, 조회 시 주차 관리 기록을 남긴다.

### 개발환경

* Visual Studio Community 2019
* Windows Forms (.NET Framework)
